# AçıqTəhsil — GitHub Pages Website

A recreation of the [AçıqTəhsil](https://mamedzadejalya.wixsite.com/hsil) website, built as a single-file static site for GitHub Pages.

## 🚀 How to deploy on GitHub Pages

1. **Create a new GitHub repository** (e.g. `aciqtehsil`)
2. **Upload `index.html`** to the root of the repo
3. Go to **Settings → Pages**
4. Under *Source*, select **Deploy from a branch → main → / (root)**
5. Click **Save** — your site will be live at:
   `https://YOUR-USERNAME.github.io/aciqtehsil/`

## 📁 Files

| File | Description |
|------|-------------|
| `index.html` | The full website (all CSS & JS included) |

## ✏️ Customization

- **Colors**: Edit `:root` CSS variables at the top of `index.html`
- **Content**: All text is in plain HTML — easy to find and edit
- **Links**: Replace the Wix links with your own pages as you build them out
